# Library render — the signature visual

The Library section renders a handcrafted CSS shelf by default. When a real
photographic/3D render of a walnut shelf with leather-bound, gold-titled books
is available, drop it here and it replaces the CSS shelf automatically.

| File | Size | Notes |
|------|------|-------|
| `breadlight-library.webp` | 2000×1200 (desktop) | Walnut shelf, leather spines, gold foil, warm light |
| `breadlight-library-mobile.webp` | 1080×1080 (optional) | Tighter crop for small screens |

Art direction: antique walnut, warm integrated lighting, realistic shadows,
gold foil titles, subtle embossing. No flat/cartoon books.
Compress to WebP ≤250 KB. Filenames must match exactly.
