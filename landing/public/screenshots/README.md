# Screenshots

Drop real phone screenshots here to replace the branded placeholders. Until a
file exists, the `<Screenshot>` component renders a labeled placeholder at the
correct phone aspect ratio — nothing ever appears broken.

## Required files (WebP, ~1080×2340 / 9:19.5)

| File | Shown in |
|------|----------|
| `home-screen.webp` | Hero, App Preview |
| `library-screen.webp` | App Preview |
| `reader-screen.webp` | Bilingual Reading feature, App Preview |
| `devotional-screen.webp` | Audio & Devotional features, App Preview |
| `vocabulary-screen.webp` | Vocabulary feature, App Preview |
| `journey-screen.webp` | Pronunciation feature, App Preview |

## Tips
- Export at 9:19.5, compress to WebP (`cwebp -q 82`), keep each under ~250 KB.
- Use real app captures — do not stage fake UI that misrepresents the product.
- Filenames must match exactly (lowercase).
