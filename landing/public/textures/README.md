# Textures

Subtle grain overlays used at low opacity for paper/leather feel. Optional —
the site ships with a CSS-generated paper grain, so these only add richness.

| File | Size | Notes |
|------|------|-------|
| `paper-grain.webp` | 512×512 tileable | Very faint, ≤30 KB, used at ~4% opacity |
| `leather-grain.webp` | 512×512 tileable | For the Library shelf, ≤40 KB |
